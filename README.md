```sh
$ some_tool -v

$ asdjw1jwhdajh1idojad # frustration
bash: asdjw1jwhdajh1idojad: command not found

$ some_tool --help

Usage:
  some_tool <flags> <args> [--parameter1 value1 --parameter2 value2 ...]

# f*cking finally!
```
